Først blir en kjent, og deretter vil tillit gradvis etableres. Det gis rom for den gode samtalen rundt bålet, eller mens vi går på tur. For våre deltagere vil det være en ny erfaring å kunne dele gode opplevelser, tanker og følelser med likesinnede. Noe som kan skape gode og varige bånd - et vennskap.

The FitnessGram PACER Test is a multistage aerobic capacity test that measures a student's aerobic capacity by having them run back and forth as many times as they can, with each lap signaled by a beep sound. It is part of the FitnessGram assessment and is used to estimate overall performance and fitness levels. The test is also known as the Progressive Aerobic Cardio Endurance Run (PACER)
